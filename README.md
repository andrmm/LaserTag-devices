# Проект имитации стрелкового оружия и приёмника для игры в лазертаг.

Проект содержит публичный и приватный репозитории. В публичном репозитории только 2 скетча: один скетч находится в папке "LaserTag-Gun", второй в папке "Lasertag-Receptor". Оба скетча предназначены для прошивания платы на базе ESP32-WROOM-32D (30 выводов). В приватном репозитории дополнительно схемы подключения и перечень радиоэлектронных элементов, необходимые для изготовления имитации стрелкового оружия и приёмника для игры в лазертаг.

## Описание
Данный проект содержит данные для изготовления системы Лазертаг. Скетч LaserTag-Gun прошивается в ESP32 в устройстве имитации стрелкового оружия, Lasertag-Receptor - в устройстве приёмника. Скетчи прошиваются в плату на базе ESP32-WROOM-32D на 30 выводов. В приветном репозитории приведены схемы электрические принципиальные электронной "начинки" для устройств, и перечень компонентов. По мере модернизации и модификации данные в публичном и приватном репозитории будут дополняться и редактирорваться.

## Использование
Для прошивки скетчей необходим персональный компьютер с установленым Arduino IDE и USB-кабель с USB-C на одном конце. В Arduino IDE в менеджере плат необходимо подключить расширение для работы с платами ESP32. Перед загрузкой прошивки убедиться в наличии библиотек "esp32-hal-ledc" и "timer", выбрать нужный COM-порт и устройство "ESP32 Dev Module".
